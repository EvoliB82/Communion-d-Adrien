# First communion website - how to open it

This folder contains the website made for Adrien's first communion.

## Best way to view it

Use a PC or laptop, not a phone.

Open it with Google Chrome.

If you press `F11` once Chrome is open, this puts the browser in full screen and makes the site looking much better.

## How to start

1. Keep all the files and folders together in this same folder.
2. Open Google Chrome.
3. Drag `index.html` into the Chrome window.
4. Press `F11` for full screen.
5. Click anywhere on the page once if the music does not start.

Another option: right-click `index.html`, choose `Open with`, then choose `Google Chrome`.

## Important

Do not move only one HTML file by itself. The website needs the `assets` folder and the MP3 files to stay next to it.

If Chrome asks for permission to play sound, allow it.

To leave full screen, press `F11` again.


# Site de la première communion - comment l'ouvrir

Ce dossier (Communion Adrien), contient un site web réalisé pour la première communion d'Adrien.

## Meilleur manière de le voir

Utilisez un PC ou un ordinateur portable, pas un mobile

Ouvrez le avec  Google Chrome

Si vous appuyez sur `F11` une fois que Chrome est ouvert, le navigateur va passer en mode plein écran, et ça donne un meilleur rendu du site.

# Comment démarrer 

1. Gardez tous les fichiers et dossiers dans le dossier original.
2. Ouvrez Google Chrome
3. Maintenez `index.html` avec le click gauche de la souris, et déposez le dans la fenêtre Google Chrome
4. Appuyez sur `F11`
5. Cliquez n'importe où sur la page si la musique ne se lance pas.

Une autre option : faites un click droit sur `index.html`, allez sur `ouvrir avec >` et selectionnez `Google Chrome`

## Imortant

Ne déplacez pas seulement un seul fichier HTML indépendamment des autres. Le fichier a besoin du dossier `assets`, des fichiers mp3 et des autres pages près de lui.

Si Chrome demande la permission de lancer un son, autorisez-le.

Pour quitter le mode plein écran, clickez sur `F11`
